# Functional Requirements

## 1. Product Catalog
The website shall display a catalog of vegetable seeds organized by the following categories:
- **Leafy Vegetables**: Spinach, Lettuce, Amaranthus, etc.
- **Root Vegetables**: Carrot, Radish, Beetroot, etc.
- **Bulb Vegetables**: Onion, Garlic, etc.
- **Stem Vegetables**: Celery, Asparagus, etc.
- **Flower Vegetables**: Cauliflower, Broccoli, etc.
- **Fruit Vegetables**: Tomato, Brinjal, Chili, Okra, etc.
- **Seed/Pod Vegetables**: Beans, Peas, etc.

Each product entry should display:
- Product Image
- Product Name
- Price (if applicable)
- "Add to Cart" or "View Details" button

## 2. Filtering Functionality
- Users shall be able to filter products by the categories listed above.
- Selecting a category should update the product grid to show only items belonging to that category.
- A "All" option should be available to view all products.

## 3. Map Integration
- The website shall include a map section (e.g., on the Contact or Home page) displaying the physical store location.
- The map should be interactive (zoom/pan) if possible, or a static link to Google Maps.

## 4. Responsive Design
- The website shall be responsive and function correctly on:
    - Desktop computers
    - Tablets
    - Mobile phones
- The layout should adjust gracefully to different screen sizes.

## 5. Navigation
- A clear navigation menu shall be provided to access different sections of the site (Home, Products, About, Contact, etc.).
